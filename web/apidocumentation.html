Path Request Type Parameters/Json Description
PERSON API:
api/person GET [{id,firstname,lastname,email,address{id,street,additionalinfo,cityinfo{zipcode,city}},phones[{number,description}],hobbies[{id,name,description}]}] Return an array of all persons in DB
api/person/{id} GET {id,firstname,lastname,email,address{id,street,additionalinfo,cityinfo{zipcode,city}},phones[{number,description}],hobbies[{id,name,description}]} Return a person with given id
api/person/create POST {firstname, lastname, email, street, additionalinfo, zipcode} Creates person and address in DB and assigns address to person
api/person/delete POST [id] Deletes person with given id from DB
api/person/address PUT {id //person, street, additionalinfo, zipcode} Updates address for person
api/person/phone POST {id //person, number, description} Creates and assign a phone to a person with given id
api/person/phone DELETE [number] Deletes phone with given number
api/person/phone/{id} GET {id,firstname,lastname,email,address{id,street,additionalinfo,cityinfo{zipcode,city}},phones[{number,description}],hobbies[{id,name,description}]} Returns person with given number
api/person/hobby/create POST {name,description} Creates a hobby with given parameters
api/person/hobby/count/{id} GET "count" Returns count of people with hobby with given id
api/person/hobby/add POST {personid,hobbyid} Add hobby with given hobbyid to person with given personid
api/person/hobby/delete DELETE {id} Delete hobby with given id from DB
api/person/hobby/person/{id} GET [{id,firstname,lastname,email,address{id,street,additionalinfo,cityinfo{zipcode,city}},phones[{number,description}],hobbies[{id,name,description}]}] Returns array of persons with hobby that contains given id
api/person/hobby PUT {personid, hobbyid} Removes a hobby with given id from person with given id
api/person/hobby GET [{id,name,description}] Returns array with all hobbies
api/person/hobby/{id} GET {id //hobby,name,description,persons[{id //person,firstname,lastname,email,address{id //address,street,additionalinfo,cityinfo{zipcode,city}},phones[{number,description}]}]} Returns hobby with given id
